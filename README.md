## âœ¨ Features

### 1. Crop Selection
- Dropdown-based crop variety selection  
- Displays:
  - Expected yield range
  - Common pests and diseases
  - Preferred soil conditions  
- Uses a mock crop database (extendable to 500+ varieties)



### 2. Yield Predictor
- Inputs:
  - Soil type
  - Weather pattern
  - Cultivated area (hectares)
- Outputs:
  - Base yield
  - Optimized yield (+20%)
  - Yield improvement
- Demonstrates research optimization benefits 



### 3. Pest & Disease Identifier
- Inputs:
  - Symptom description
  - Image upload (for future AI integration)
- Rule-based identification (demo logic)
- Displays likely pest or disease with explanation



### 4. Soil Analysis
- Inputs:
  - pH level
  - Nitrogen, Phosphorus, Potassium values
- Provides recommendations for:
  - pH correction
  - Nutrient supplementation
- Helps improve soil fertility management


### 5. Climate Adaptation
- Inputs:
  - Average temperature
  - Annual rainfall
- Suggests climate-appropriate crops
- Encourages climate-resilient farming decisions


### 6. Sustainable Practices
- Informational cards on:
  - Crop rotation
  - Organic fertilizers
  - Water conservation
  - Integrated Pest Management (IPM)
- Promotes environmentally friendly farming
